# AI-Verify-Pro 專為繁體中文語境設計的 AI 文案偵測工具。
AI-Verify Pro：AI 文案特徵深度檢測器AI-Verify Pro 是一個基於 Python 與 Streamlit 開發的文字分析工具。它不依賴黑箱模型，而是透過 「多維度特徵權重」 與 「統計學分析」 來識別文字是否具備典型的 AI 生成痕跡。🌟 核心特色組合特徵門檻 (Combo Threshold)：獨創邏輯，當多種 AI 修辭技巧同時出現時才觸發加重計分，有效降低對真人感性寫作的誤判。Emoji 密度與結構分析：精準識別 AI 常用的「清單式圖示」與「功能型錨點」排版模式。段落節奏統計：利用標準差計算段落長度起伏，揪出 AI 過於整齊、缺乏靈魂的寫作節奏。真人氣息補償機制：內建台灣在地口語、語助詞偵測，為具備溫度的真人創作提供得分加成（減分）。🛠️ 技術棧Python 3.9+Streamlit：網頁互動介面Statistics & Regex：核心分析演算法🚀 快速開始 (Windows 使用者)本專案已內置自動化腳本，方便使用者快速部署環境。
1. 執行運行腳本找到資料夾中的 run_app.bat ，點擊兩下：腳本會自動檢查並安裝所需的依賴套件（如 streamlit）。完成後會自動開啟瀏覽器並啟動偵測器網頁。手動啟動方式：如果你偏好使用命令列，可以輸入：
Bashpip install -r requirements.txt
streamlit run your_app_name.py

90+🚨 AI 原始模板具備極高密集的 AI 標籤，幾乎確定為機器直接生成。
60 - 89🚩 確定為 AI 生成具備明顯的 AI 骨架與組合套路。
40 - 59⚠️ 高度 AI 參與人機協作特徵明顯，核心排版邏輯仍為 AI 風格。
20 - 39💡 疑似 AI 潤飾以真人寫作為主，但使用了部分 AI 的修辭習慣。
0 - 19✅ 表現自然特徵零星，符合人類自然的寫作慣性。< 0☘️ 純真人手寫完全避開機器邏輯，充滿個人情緒與口語特徵。
